如何用hugo搭建个人博客

hugo的下载和配置

1.下载hugo 2.将下载下来的包解压缩后，放到"D:\Software\hugo”,此目录下只有"hugo.exe"文件。 3.配置环境变量，以便在全局使用hugo。

4.重启终端，运行hugo version命令，查看版本，见到下图说明成功。

看hugo文档快速搭建个人博客

1.进入Hugo官网，点击快速开始（quick start）

2.从Step2开始抄代码在终端运行，直至Step7（此文档步骤叙述十分详细，在此不再赘述）

3.在/public的上级目录中创建”.gitignore”，在此文件中将/public写入进去，这样public文件夹将单独成为一个不受影响的独立仓库。

4.cd进入/public文件夹，运行如下代码

git init
git add .
git commit -m "提示"  //亦可（git commit -v）

5.在GitHub上创建一个仓库名字 bobolife.github.io (bobolife为我的GitHub账户名)，将本地仓库（public）和远程仓库关联起来。

运行下面两行代码（你的新建仓库会有，直接复制粘贴运行就行）

ps：

1.在生成网站目录下（public的父级目录），运行

hugo new posts/第二篇博客.md

2.运行 hugo -D，得到新的 public 目录

3.进入 public 目录，执行一下操作

git commit -m update
git push -f 其中 -f 是强制上传的意思

4.等待片刻，博客会出现第二篇文章
